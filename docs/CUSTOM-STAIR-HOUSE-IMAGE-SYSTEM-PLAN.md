# Custom Stair House Image System Plan

## Purpose
Create one clean image workflow so Custom Stair House can:
- improve website trust and conversion
- build a useful gallery
- support Instagram posting
- avoid random photo chaos later

The website should be the **source of truth for conversion**.
Instagram should be a **supporting proof/distribution channel**.

---

## Core Rule
Use **one master image library** and publish from it to:
1. website pages
2. gallery
3. Instagram
4. sales materials / dealer packet

Do **not** make Instagram the primary image system for the website.

---

## Folder Structure
Use a main folder in Google Drive or another shared storage location:

`Custom Stair House / Images /`

Inside it:

### 01_RAW_ORIGINALS
Original unedited photos and videos.

Subfolders:
- `installed-jobs`
- `product-closeups`
- `before-after`
- `dealer-samples`
- `shop-process`
- `team-or-founder`

### 02_BEST_SELECTS
Only the strongest usable photos.

Subfolders:
- `website-shortlist`
- `instagram-shortlist`
- `dealer-packet-shortlist`

### 03_WEBSITE_READY
Compressed/resized images prepared for web upload.

Subfolders:
- `homepage`
- `stair-treads`
- `stair-nosings`
- `vent-covers`
- `gallery`
- `for-flooring-stores`

### 04_INSTAGRAM_READY
Cropped/exported images for feed posts, carousels, or stories.

### 05_ARCHIVE
Old, weak, duplicate, or unused images.

---

## Image Categories to Collect

### 1. Installed Project Photos
Highest value.

Goal:
Show that the finished result looks clean in a real space.

Best examples:
- full staircase view
- finished tread installation
- finished nosing in context
- floor + stair transition match
- vent cover installed in room

### 2. Product Close-Ups
Show craftsmanship and match quality.

Best examples:
- edge detail
- grain alignment
- texture match
- finish consistency
- vent cover detail
- nosing profile detail

### 3. Before / After Examples
Very strong for trust.

Goal:
Show what the problem looked like before and how the custom match improved it.

Examples:
- generic or mismatched accessory vs final matched version
- unfinished stair edge vs completed look
- plain vent vs flush matched vent

### 4. Dealer Sample Photos
Useful for dealer page and sales support.

Best examples:
- samples displayed neatly
- sample box or dealer kit
- multiple finish examples
- showroom-ready presentation

### 5. Shop / Process Photos
Supportive, not primary.

Examples:
- material being worked on
- fabrication in progress
- finishing process
- founder/shop authenticity

### 6. Founder / Team Photos
Needed lightly for trust.

Examples:
- founder in workshop
- founder with product
- clean professional shot at work

### Priority 1 — Homepage
Upload first:
- 1 strong hero-quality image
- 2 to 4 proof images

Best homepage image types:
- finished stair tread project
- close-up showing exact-match quality
- finished nosing detail
- high-quality installed vent cover photo

Goal:
The homepage should immediately feel real, not text-only.

### Priority 2 — Product Pages

#### Stair Treads Page
Use:

- full staircase shots

#### Stair Nosings Page

#### Vent Covers Page

- flush-mount vent examples

### Priority 3 — Gallery Page

- installed jobs

### Priority 4 — For Flooring Stores Page

---

## Minimum First Upload Target
Do not wait for a perfect giant library.

Start with:
- 3 homepage-quality images
- 3 stair tread images
- 3 stair nosing images
- 2 vent cover images
- 4 to 8 gallery images
- 2 dealer/sample photos

That is enough to make the site feel much more credible.

---

## Image Selection Rules
Choose images that are:
- bright
- sharp
- uncluttered
- clearly relevant
- easy to understand on mobile
- proof-oriented, not random

Avoid images that are:
- blurry
- dark
- too far away
- duplicate angles
- distracting background heavy
- hard to interpret without explanation

---

## Naming Convention
Use clean filenames for website images.

Examples:
- `custom-stair-tread-northern-virginia-01.jpg`
- `exact-match-stair-nosing-detail-01.jpg`
- `flush-mount-vent-cover-wood-floor-01.jpg`
- `flooring-store-sample-display-01.jpg`

This helps organization and SEO.

---

## Basic Website Image Specs
Use web-optimized versions.

Recommended:
- JPG or WEBP for photos
- compressed for speed
- large enough to look sharp
- consistent crop style where possible

Practical target:
- homepage / featured images: around 1600px wide
- gallery/support images: around 1200px wide
- thumbnails smaller as needed

---

## Alt Text Rules
Each website image should have useful alt text.

Examples:
- `Exact-match custom stair tread made from customer-provided wood flooring`
- `Custom stair nosing detail matching engineered hardwood floor`
- `Flush mount wood vent cover matched to finished floor in Northern Virginia home`

Do not keyword stuff. Just describe the image clearly.

---

## Instagram Role
Instagram should be used to:
- publish finished work consistently
- build visual proof over time
- make the business look active and real
- create a second trust layer for visitors who check socials

Instagram should **not** be the main gallery backend.

---

Start with:
1. best finished stair project
2. exact-match nosing close-up
3. vent cover before/after
4. sample/dealer kit photo
5. workshop/process photo
6. carousel showing problem → solution → final result

## Best Immediate Next Step
When ready to collect images, do this in order:
1. gather all existing photos into one folder
2. sort into categories
3. choose the best 15 to 25 images
4. shortlist the best 8 to 12 for the website
5. prepare web-ready exports
6. upload homepage + product-page images first
7. use the same asset pool for Instagram
